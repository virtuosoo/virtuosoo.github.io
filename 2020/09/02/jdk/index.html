windows下jdk环境变量配置

发表于 2020-09-02

考完试想学下java,第一步就是要装jdk， 但网上各种环境变量的配置教程好多都不管用了，找了半天终于找到一个成功的。

1.在系统变量中创建JAVA_HOME变量，路径为自己安装jdk的位置，比如我的是C:\Program Files\Java\jdk1.8.0_261。

2.在系统变量中创建classpath变量，路径为.;%java_home%\lib。

3.在系统变量的已有的Path变量中添加%java_home%\bin。

这样就ok了。

可以在cmd中输入java java -version javac -version等命令测试是否配置成功，如果出现很多奇怪的东西就说明成功了。
